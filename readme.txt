An open source Unity asset package for VR that hides text until you gaze at it. Usage can be seen in this video: https://youtube.com/shorts/g_lD1Q7jhb0.

The included Unity package contains a base scene that contains the curvedgazemonitor application. You will need to do the following:

Import SteamVR (via the Unity Package Manager)
Import the SRanipal unitypackage (https://developer.vive.com/resources/vive-sense/eye-and-facial-tracking-sdk/download/latest/) under SDK->Unity
Download and install the SRanipal runtime (https://forum.htc.com/topic/5642-sranipal-getting-started-steps/)
